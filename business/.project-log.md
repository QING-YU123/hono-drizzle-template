# 项目执行日志

这个文件记录了项目的所有执行历史，由 **project-orchestrator** skill 自动维护。

## 使用说明

### 如何开始
在 Claude Code 中执行以下任一命令：

```
/project-orchestrator
```

或直接说：
- "开始新版本开发"
- "查看项目进度"
- "继续开发"
- "下一步做什么"

### 日志格式

每次执行都会记录以下信息：
```markdown
## YYYY-MM-DD HH:mm - {阶段名称}

**执行 Skill**: {skill_name}
**版本**: {version}
**状态**: {success/failed/in_progress}
**输出**: {output_files}
**用户反馈**: {user_feedback}
**下一步**: {next_action}
```
