**文件**: `src/utils/jwt.ts`
**行号**: 7

**问题描述**:
JWT 密钥使用默认值，生产环境可能忘记设置。

**当前代码**:
```typescript
const JWT_SECRET = process.env.JWT_SECRET || 'default-secret-key-change-in-production';
```

**修复建议**:
```typescript
const JWT_SECRET = process.env.JWT_SECRET;
if (!JWT_SECRET) {
    throw new Error('JWT_SECRET environment variable is required');
}
```
