### 1. 参数验证逻辑修复

**问题**: 注册和登录时，空字符串密码验证顺序错误

**修复代码** (`src/controllers/auth.controller.ts`):
```typescript
// 修复前
if (typeof password !== 'string' || password.length < 8) {
    return c.json({ message: 'password must be at least 8 characters' }, 400);
}

// 修复后
if (typeof password !== 'string' || password.trim().length === 0) {
    return c.json({ message: 'password is required' }, 400);
}
if (password.length < 8) {
    return c.json({ message: 'password must be at least 8 characters' }, 400);
}
```

### 2. 测试数据隔离改进

**问题**: 固定邮箱地址导致多次运行测试时数据冲突

**修复方案**: 使用时间戳生成唯一邮箱地址
```typescript
const generateEmail = (prefix: string): string => {
  const timestamp = Date.now();
  const random = Math.floor(Math.random() * 1000);
  return `${prefix}-${timestamp}-${random}@example.com`;
};
```

### 3. 测试执行顺序优化

**问题**: 刷新令牌测试中，beforeAll 钩子获取的令牌被第一个测试使用，导致后续测试失败

**修复方案**: 将 `beforeAll` 改为 `beforeEach`，确保每个测试都有新的令牌

### 4. 测试超时调整

**问题**: bcrypt 密码哈希操作耗时较长，导致超时

**修复方案**: 为密码相关测试增加超时时间到 15 秒
