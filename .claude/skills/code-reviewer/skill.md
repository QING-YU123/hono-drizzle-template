# 代码审查器

这个技能帮助你在测试阶段结束后，对项目代码进行全面审查，生成符合 AI 驱动的代码审查标准与报告规范的详细审查报告。

## 🎯 技能概述

### 主要功能
- 📊 **版本识别**：识别当前项目版本，读取对应版本的测试报告
- 📖 **代码分析**：全面分析代码变更，包括新增、修改的文件
- 🔍 **多维审查**：从7个维度进行系统性代码审查
- 📝 **报告生成**：生成结构化、详细的代码审查报告
- 💡 **改进建议**：提供可操作的修复建议和最佳实践

### 审查维度
1. **代码风格与规范** - 检查代码风格一致性、命名规范、注释完整性
2. **潜在 Bug 与逻辑错误** - 识别未处理的错误、空值访问、类型安全问题
3. **性能影响** - 分析数据库查询、计算复杂度、资源使用
4. **安全性** - 检查注入风险、XSS漏洞、敏感数据暴露
5. **可维护性与架构** - 评估代码复杂度、重复代码、架构一致性
6. **测试覆盖** - 分析测试用例质量和覆盖范围
7. **变更影响范围** - 评估破坏性变更和文档更新

### 前提条件
- ✅ 测试阶段已完成，测试报告已生成
- ✅ `business/iterations/{version}/测试/测试报告/` 目录存在
- ✅ 项目代码已完成编写
- ✅ 依赖已安装，项目可正常运行

## 🔄 完整工作流程

### 第一阶段：版本和测试报告识别（必做）

#### 1. 识别当前版本

- 确认测试报告已生成

### 第五阶段：问题修复（可选）

当用户选择修复发现的问题时，执行以下流程：

#### 1. 修复严重问题

**常见问题及修复方案**：

**问题 A: 参数验证逻辑顺序错误**
```typescript
// ❌ 错误：空字符串会触发长度验证而非必填验证
if (typeof password !== 'string' || password.length < 8) {
    return c.json({ message: 'password must be at least 8 characters' }, 400);
}

// ✅ 正确：先检查空字符串，再检查长度
if (typeof password !== 'string' || password.trim().length === 0) {
    return c.json({ message: 'password is required' }, 400);
}
if (password.length < 8) {
    return c.json({ message: 'password must be at least 8 characters' }, 400);
}
```

**问题 B: 测试数据隔离问题**
```typescript
// ❌ 固定邮箱导致数据冲突
const email = 'test@example.com';

// ✅ 使用时间戳生成唯一邮箱
const generateEmail = (prefix: string): string => {
  const timestamp = Date.now();
  const random = Math.floor(Math.random() * 1000);
  return `${prefix}-${timestamp}-${random}@example.com`;
};
```

**问题 C: 测试执行顺序问题**
```typescript
// ❌ beforeAll 导致令牌被第一个测试使用
describe('POST /auth/refresh', () => {
  beforeAll(async () => {
    // 获取令牌...
  });

  it('TC-001: 第一次使用', async () => {
    // 使用令牌...
  });

  it('TC-002: 第二次使用失败', async () => {
    // 同一令牌已失效...
  });
});

// ✅ beforeEach 确保每个测试有新的令牌
describe('POST /auth/refresh', () => {
  beforeEach(async () => {
    // 每个测试前重新获取令牌...
  });

  it('TC-001: 刷新成功', async () => {
    // 使用新令牌...
  });

  it('TC-002: 重复使用失败', async () => {
    // 使用新令牌...
  });
});
```

**问题 D: 测试超时问题**
```typescript
// bcrypt 操作耗时较长，需要增加超时时间
it('密码验证测试', async () => {
  // 测试代码...
}, 15000); // 增加超时到 15 秒
```
